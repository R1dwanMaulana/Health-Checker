## Setup

```bash
# install dependencies
$ yarn # Or npm install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start

# build for firebase and launch server
$ yarn build:firebase
$ yarn start:firebase

# generate static project
$ yarn generate

# Run unit test
$ yarn test

# Run e2e test
$ yarn dev
$ yarn test:e2e
```
